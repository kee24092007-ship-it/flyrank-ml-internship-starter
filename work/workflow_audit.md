## 3. Target Tasks & Success Definitions

### Task 1: Debugging Python/ML syntax errors
* **"Done Well" Means:** The code executes without syntax errors, follows clean PEP8 formatting, and I fully understand what caused the bug.

### Task 2: Summarizing long blog posts or documentation
* **"Done Well" Means:** The summary lists key terms, code examples, and mathematical equations accurately without hallucinating information.

### Task 3: Generating dummy datasets for coding practice
* **"Done Well" Means:** The mock dataset contains realistic values, has no missing entries, and is ready to load into a pandas DataFrame instantly.
